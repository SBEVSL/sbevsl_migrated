ConMOL Installation Instructions

download PyMOL from http://pymol.sourceforge.net and install to default location

Unzip all ConMOL files to:  C:\Program Files\DeLano Scientific\pymol\modules\pmg_tk\startup\

When all files are in place, the ConMOL interface can be run by choosing ConSCRIPT from the PyMOL plug-in menu.
