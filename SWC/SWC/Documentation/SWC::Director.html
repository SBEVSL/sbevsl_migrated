NAME

SWC::Director - main object in the SWC package

SYNOPSIS

    use SWC::Director;
    my $object = SWC::Director->new();
    $object->set_input_converter("SBEVSL");
    $object->set_output_converter("pymol");
    $object->convert(\@lines);
    my @output = @ { $object->get_output() };
    foreach (@output) { 
        print "$_\n";
    }

DESCRIPTION

This module is the main object in the SWC package. This object is the only object that is manually created by the user. This object contains methods to create and control input and output converters.

Methods

new
Returns a new SWC::Director object with no converters defined.

set_input_converter
Sets the input converter type to the input language.

set_output_converter
Sets the output converter type to the desired output language.

get_output
Retrieves the output from the output converter.

Returns an array of strings that represent the final conversion.

convert
This method calls the convert methods in the input converter and then the output converter.
