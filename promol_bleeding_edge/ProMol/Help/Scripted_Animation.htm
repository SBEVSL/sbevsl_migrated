Scripted Animation

The Scripted Animation group allows a user to quickly create a movie which can turn or movie an object over the number of indicated.

How to make a scripted animation:

1) Set the number of frames that you want to have in your movie by entering the number into the Frames in Movie entry field and clicking the Set Frames button.

2) Enter an integer value into an entry field and click the button which corresponds to the desired function. If you want to perform more than one function at a time you can use the Turn All, Move All, or Do All button to have a frame with multiple functionality.

The Frame entry field can be manually set; however, if the frame number is higher than the number of frames set in Frames in Movie entry field, these frames will not be displayed.
