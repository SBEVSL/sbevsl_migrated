Selection Controls

 

The selection Controls group allows users to more eaisly manipulate objects within the PyMol viewer.

The Mask button masks the object indicated int the Name Entry field and prevents it from being manipulated.

The Unmask button unmasks the object indicated int the Name Entry field and allows it to be manipulated.

The ProtectAll Except button protects all objects except the object indicated in the Name Entry field and prevents them from being manipulated.

The Protect button protects the object indicated int the Name Entry field and prevents it from being manipulated.

The Deprotect button deprotects the object indicated int the Name Entry field and allows it to be manipulated.

The Protect All Except button protects all objects except the object indicated in the Name Entry field and prevents them from being manipulated.
