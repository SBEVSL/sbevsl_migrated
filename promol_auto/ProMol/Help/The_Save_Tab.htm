The Save Tab

Here you'll find two buttons that you can use to save your work.  The Save Session button allows you to save all you work up to a specific point.  This includes all the changes you have made to the appearance of your molecule so that you can continue working on it later and pick up right where you left off.The Save Image button will save PNG image of the molecule that is currently being displayed in the PyMOL viewing window.  You will also have the option of ray tracing the image, which increases its overall quality.
