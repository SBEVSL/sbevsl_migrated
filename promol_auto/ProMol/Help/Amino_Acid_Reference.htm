The Amino Acid Reference group

 

The amino acid reference tool is useful when trying to find out amino acid codes as well as the structure of individual amino acids.

The Amino Acid Code entry field allows a user to enter a one or three letter code (case sensitive, i.e. a or ala for alanine) and return the amino acid name and structure after clicking the Submit button.

The Amino Acid Name entry field allows the user to return the structure and code for the amino acid entered. Again, this is case sensitive so capitalizations will not work

The 2D and 3D radio buttons allow the user to toggle between a 2D structure and an 3D structure. Once a change has been made to the radio button, the amino acid code or name must be resubmitted for the new structure to appear.
