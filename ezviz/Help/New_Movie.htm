New Movie

To create a new movie you must first load a PDB file into PyMOL using either the 'Fetch PDB' or 'Open' button.  Once the molecule is loaded, push the New Movie button.  This will enable the rest of the buttons on the interface which can then be used to make a movie. 

 

The movie making process can be thought of as having two phases: adding frames and editing frames.  The adding frames phase is like the filming stage where the molecle's actions are recorded.  Once the frames are recorded, you can then edit them by altering their colors or structural representations. These phases must be completed in order, with the addition of frames preceeding editing.s

Adding Frames

The 'Add Frames' box in the middle of the interface contains all the buttons and menus needed to add frames to the movie.  First you must select one of six actions using the radiobuttons on the left side of the box. The six actions that can be added are:

  • Zoom In
  • Zoom Out
  • Rotate Y 180
  • Rotate X 180
  • Rotate Z 180
  • Hold on View

If you select the zoom in option, you must also select a feature from the zoom in on dropbox.

 

Once you have selected an action, you must then select the number of frames for that action from the Frames dropdown.  The number of frames corresponds to the duration in which you want the action to occur over.  By default, 15 frames are played per second so to make a 1 minute movie you would have to add 900 frames.  The frame speed can be adjusted using the radiobuttons on the 'Movies' tab.

 

After selecting both an action and its duration, you can then hit the Add Frames button to append the frames to the end of the movie.  If you make a mistake, or do not like an addition that you have made you can click the Undo button to remove the frames from your movie.

 

When you have completed adding frames, you must then click the Make Movie button to stitch the movie together.

Editing Frames

Once the movie is made, you can then edit the individual frames of the movie.  This includes changing colors and structural representations. Click the Edit Frames button to open the Frame Editor window. 

 

 

Once the frame editor window is open, you can navigate to the frame you want to edit using the slider on the bottom of the window.  Dragging the slider will change the position of the molecule in the viewing window to correspond to the selected frame.  After navigating to the correct frame, use the Select dropdown to choose the molecular feature that you want to alter.  This list includes: bound ligands, individual chains, potential active sites, and the orignal list of features found on the Commands tab.  If you select a bound ligand, you can then use the radiobuttons to the right of the Select menu to include so additional area surrounding the ligand.  You can choose to include the area within 1, 3, or 5 angstroms of the selected ligand. 

 

After selecting a feature from the Select menu, use the dropdowns in the Options box to alter the appearance of the structure. Use the Color dropdown to change the color of the selected feature and use the Hide/Show dropdowns to show and hide different strucural representations.  Additionally, the Presets dropdown can be used to apply one of the Ez-Viz preset views to the entire molcule. 

 

Upon making your selections a written confirmation will be displayed in the Current Edit box.  If you agree with what is displayed in the Current Edit box, click the Accept button for the changes to be made to molecule.  If you disagree with the written confirmation, repeat your dropdown selections and verify that the correct edit is displayed in the Current Edit box. When the Accept button is pressed, the edits will be printed to the text box on the right side of the Frame Editor window to provide you with a visual list of all the edits made to a specific frame.  To make additional edits to the same frame, repeat the steps above.  If you make a mistake, or do not like a change you have made you can click the Undo button to remove the edit.  This will remove the edit from the text box and from being potentially incorporated into you movie.

NOTE: Using the Undo button will not revert the molecule back to the way it looked prior to the edit being made. 

 

Once you have completed all the necessary changes to a given frame, you must then push the Frame Complete button.  This step is necessary to incorporate the changes into the movie! Just because the changes are present in the viewing window, does not a guarantee that they have been incorporated into your movie. 

NOTE: It is important to keep in mind that only one edit can be applied at a time!  For example you cannot change the color of a bound ligand and show sticks for the same bound ligand in the same step.  Each edit must be made independently of the other!  

 

When all your edits are complete, you DO NOT need to click the Make Movies button.  Any edits will have already been incorporated into the movie when the Frame Complete button was pressed.

   

The Preview Button

At any time during the movie's creation you can push the Preview button to view the current progress of your movie.

 

The Make Movie Button

When you have finished appending frames to your movie you must then hit the Make Movies button to stitch your movie together.  After clicking this button your movie is complete and can be played back using the Play button on the Movies tab.
