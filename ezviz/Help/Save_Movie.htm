Save Movie

Pushing the 'Save Movie' button allows users to save a completed movie to file so that it can be played or worked on at a later time.  When the Save File dialog window opens specify the location where you would like the movie to be saved and a name for the file. Both complete and incomplete movies can be saved.  Complete movies are those that have had no changes after the Make Movies button has been pressed, while incomplete movies have.
