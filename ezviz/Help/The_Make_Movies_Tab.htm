The Make Movies Tab

The Make Movies tab was developed in 2007 to provide users with extended capabilities for movie making in PyMOL.  Included in this tab are tools for creating a new movie, loading a previously made movie, and saving created movies for later playback.

 

NOTE: In order for the Make Movies tab to function correctly, when installing EZ-Viz Movie, the user must also install the latest version of the Catalytic Site Atlas database.    This file can be retrieved from the the following website:

http://www.ebi.ac.uk/thornton-srv/databases/CSA/  

The .dat file must be place in the same directory as the source code for the interface to insure correct functionality.
