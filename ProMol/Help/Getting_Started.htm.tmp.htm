Getting Started

Installation

To use this program you must have Delano Scientific's molecular visulization program, PyMOL, installed on your computer.  For instructions on how to do this, please consult the PyMOL Manual .  Their list of system requirements and installation instructions can be found at the PyMOL installation page. 

 

Once you have PyMOL installed, launch the program to ensure that it is working properly.  You should see something much like the image shown below.

 

Now that PyMOL is installed and working properly you need to download the EZ-Viz plugin.  This can be obtained from www.xxxx.com  Save the files to an easy to remember directory.  Now launch PyMOL like you did before.  Along the top menu bar of the external GUI, you should see a menu titled Plugin.  Click this menu and find the option named 'Install Plugin'.  When the 'Open File' window pops up, navigate to where you saved the EZ-Viz plugin files and select them.  A window will appear telling you the installation was successful and that you need to restart PyMOL.  After the restart, naviagte to the Plugin menu in the External GUI.  Upon clicking, you should see a EZ-Viz option in the menu.  Select it to start EZ-Viz. A new widow should open that looks like the one below.  Now you're ready to start using EZ-Viz.
